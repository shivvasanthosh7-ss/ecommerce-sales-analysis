
E-Commerce Sales Analysis

Project Objective

Analyze e-commerce sales data to identify high-performing categories, regional trends, and profitability patterns.

---

Tools Used

* Microsoft Excel
* Google Sheets
* Pivot Tables
* Data Visualization

---

Dataset

Superstore sales dataset containing:

* Region
* Category
* Sales
* Profit

---

Key Analysis Performed

* Category-wise sales analysis
* Region-wise performance comparison
* Profit trend evaluation
* Pivot table summarization

---

Key Insights

* Technology category generated the highest sales
* Office Supplies showed stable performance
* Some regions had negative profit despite good sales
